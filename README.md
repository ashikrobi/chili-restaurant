## Installation

Run these commands after cloning the repo.

```bash
  npm install
  cd my-project
```

## Run Locally

Clone the project

```bash
  git clone https://github.com/ashikrobi/chili-restaurant
```

Go to the project directory

```bash
  cd chili-restaurant
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
